# book-eli5

> Books explained in my own words (ELI5 style)

어려운 책도 나만의 언어로, 동료에게 막힘없이 설명할 수 있을 때까지 정리합니다.

## 이 저장소의 학습 방식

**파인만 기법(Feynman Technique)** 에서 아이디어를 얻어 나만의 방식으로 적용합니다.

> "어떤 개념을 어린아이에게 설명할 수 있을 만큼 단순하게 만들지 못했다면,  
> 그 개념을 충분히 이해하지 못한 것이다."

### 4단계 프로세스

1. **읽고 줍기** - 섹션을 읽으면서 중요하다고 느낀 개념들을 가볍게 적는다
2. **내 말로 바꾸기** - 책의 문체가 아닌, 내가 평소 말하는 방식으로 다시 쓴다
3. **막히면 돌아가기** - 쓰다가 막히거나 애매한 부분 = 아직 이해 못한 부분. 다시 읽는다
4. **다듬기** - 자신있게 설명할 수 있을 때까지 단순하게, 명확하게 고친다

단순 요약이 아닙니다.  
**설명하다 막히면 → 다시 읽고 → 다시 쓰는** 과정을 반복합니다.
